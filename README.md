# MazeSolving

Simple project for implementing BFS Lee-Algorithm ([Wiki reference](https://en.wikipedia.org/wiki/Lee_algorithm))

## How to use:

1) Get your ASCII maze in a file `maze_input.txt`.

### Input design:
* ` ` - empty place.
* `#` - a wall;
* `S` - a start point;
* `F` - a finish point.
